# Prompt final — Identidad Maestranza Modelo PRO

Actúa como director de marca y diseñador senior de producto industrial. Trabaja sobre Maestranza Modelo PRO, plataforma operacional conjunta de Boiler Comp S.A. y Soldesp S.A. para planificación, órdenes de trabajo, HH, Carta Gantt, avance desde terreno, documentos y HSEQ.

## Objetivo

Aplicar una identidad digital institucional que reconozca a ambas empresas con igual peso, mantenga la lectura rápida en taller y no confunda colores corporativos con estados operacionales.

## Reglas obligatorias

1. El nombre visible del software es **Maestranza Modelo PRO**.
2. Boiler Comp y Soldesp son marcas operadoras; ningún logo reemplaza o subordina al otro.
3. Usar los archivos aprobados de `public/brand/partners/` sin deformar, recolorear, recortar ni añadir efectos.
4. En co-branding, mantener igual peso visual mediante altura óptica, espacio y divisor neutral.
5. Usar navy Soldesp como base del producto; rojo Boiler solo dentro de la placa de marca o para alertas semánticas inequívocas.
6. Mantener el naranja Soldesp como acento institucional secundario, no como reemplazo del dorado operativo.
7. Mostrar logos completos en acceso y una firma compacta en encabezados.
8. Incluir el nombre definitivo en metadatos, navegación, reportes PDF, accesibilidad y documentación.
9. Conservar responsive móvil, foco visible, contraste WCAG AA y movimiento reducido.
10. No declarar cierre sin typecheck, lint, pruebas, build y revisión visual.

## Criterios de aceptación

- El usuario identifica “Maestranza Modelo PRO” antes de ingresar.
- Boiler Comp y Soldesp aparecen juntos, legibles y sin distorsión.
- Ninguna pantalla visible conserva la denominación anterior.
- Los reportes exportados usan el nombre definitivo.
- La identidad no reduce la claridad de alertas HSEQ ni estados de producción.
